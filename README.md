# Referent

Минимальное приложение на Next.js

## Установка

```bash
npm install
```

## Запуск

Разработка:
```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

Сборка для production:
```bash
npm run build
```

Запуск собранного проекта:
```bash
npm start
```
