## Инструкция по добавлению:

1. Откройте проект на Vercel: https://vercel.com/dashboard
2. Выберите проект **referent**
3. Перейдите в **Settings** → **Environment Variables**
4. Добавьте каждую переменную:
   - Нажмите **Add New**
   - Введите **Key** (имя переменной)
   - Введите **Value** (значение)
   - Выберите окружения: ☑️ Production, ☑️ Preview, ☑️ Development
   - Нажмите **Save**
5. После добавления всех переменных:
   - Перейдите в **Deployments**
   - Найдите последний деплой
   - Нажмите на **⋯** (три точки) → **Redeploy**
   - Подтвердите перезапуск

## Проверка:

После перезапуска деплоя проверьте работу функции "Иллюстрация". Она должна работать без ошибок.
