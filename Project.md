
Приложение получает на входе адрес англоязычной
статьи, выполняет парсинг и, по соответствующей кнопке,
подключает AI и выполняет нужное действие.
Интерфейс:
1. Поле ввода URL англоязычной статьи.
2. 3 кнопки: «О чем статья?», «Тезисы», «Пост для
Telegram».
3. Блок для отображения результата, в котором
происходит генерация нужного вида ответа.
